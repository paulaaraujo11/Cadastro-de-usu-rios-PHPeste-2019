CRUD feito em PHP puro com padrão MVC durante workshop realizado no PHPeste 2019 em Recife, ministrado pelo professor Alessandro Feitoza.
Disponível do PHP 7 a cima.

Passos para reproduzir

1.Coloca o arquivo na pasta C:\xampp\htdocs

2.Percorre pelo cmd até a pasta public dentro do arquivo

cd \xampp\htdocs\phpeste\public

3.Executa o seguinte comando => "php -S localhost:8080"

4.Executar no navegador

localhost:8080/index.php
